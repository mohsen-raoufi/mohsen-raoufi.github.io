## üöÄ Functional Summary

**TrustMyETA** is a travel time prediction system that calculates a **"Trust Score"** for any given route. Instead of a single ETA, it provides a probabilistic arrival time range based on risk factors.

*   **Core Logic**: Uses **Monte Carlo Simulation** (1000 iterations) to model travel variability.
*   **Input Vectors**: Analyzes Route Geometry (tortuosity), real-time Weather (snow, visibility), and Traffic patterns.
*   **Output**: A normalized Reliability Score (0-100%), P95 (worst-case) arrival time, and categorized risk factors (Safe/Caution/High).
*   **Demo Mode**: Features a fail-safe "Demo Mode" that serves pre-calculated mock data for specific presets (Berlin, Moscow, LA) to demonstrate functionality without a live backend connection.
